Step 1: Install Node.js (v22+)

Go to the official Node.js website: https://nodejs.org/en/download/

Look for the “LTS” version (Long-Term Support). Make sure it’s Node 22.x or later.

Example: v22.17.0

Download the Windows Installer (.msi) and run it.

Keep all default options checked (just click Next → Next → Finish).

Step 2: Verify Node.js and npm

Open Command Prompt (press Win + R, type cmd, hit Enter).

Check Node version:

node -v

Should print something like v22.17.0.

Check npm version:

npm -v

Should print a number like 11.x.x.
